# Typing Test - Baka Edition!

A fun, anime-themed typing test application built with React and Vite. Test your typing speed, get roasted for your performance, and see if you can achieve the rank of "Keyboard God"!

![Demo Screenshot](https://res.cloudinary.com/dymxdpunk/image/upload/v1782412119/2026-06-25_cmqw0w.png)

## 🚀 Features

-   **WPM & Accuracy:** Measures your typing speed in Words Per Minute (WPM) and calculates your accuracy.
-   **Dynamic Feedback:** Receive a unique grade, title, and a custom roast based on your typing performance. Are you a "Turtle" or a "Demon"?
-   **Keyboard-Only Interface:** A pure, uninterrupted typing experience with a blinking caret to guide you. No mouse needed!
-   **Modern UI:** A clean and responsive design that supports both light and dark modes.
-   **Randomized Texts:** A wide variety of sentences to keep the challenge fresh.

## 🛠️ Built With

*   [React](https://reactjs.org/) - The web framework used.
*   [Vite](https://vitejs.dev/) - Next-generation frontend tooling.

## 🏁 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js and npm installed on your machine.
*   npm
    ```sh
    npm install npm@latest -g
    ```

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/ronen6999/Typing-test.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```
3.  Run the development server
    ```sh
    npm run dev
    ```
The application will be available at `http://localhost:5173` (or another port if 5173 is busy).
